爱妃

发表于 2019-07-19 | 分类于 只道寻常 |

高三某天放学后，一起去吃饭。


我心情颇佳，想到一个捉弄人的问题。哈哈，于是我一脸贼样的凑近R智，满眼冒光道：
“达令啊，如果你是皇帝，我是不是你的爱妃呀？”

“不是。”

咦，如果他说是，皇帝后宫佳丽三千，我却只是一个爱妃，赤裸裸的轻视我啊！那我可不依！可没想到，这坑竟然给他绕过去了，我一下来了兴致,追问道：

“为什么呀？”

这厮一脸文静地抬头看着我，慢斯条例的开口道：

“因为...
  你只是我的妃，不是爱妃。”




呵呵呵呵呵呵呵呵........

2015年8月29日（初来金陵）

发表于 2019-07-19 | 分类于 时间都知道 |

看到你发来的视频以前我一点也不觉得我想你。

八月底，你即将开学，正和家人在南京游玩。
每天晚上都和我聊天，告诉我你这一天里干了些什么。人真是难以捉摸，尤其是自己喜欢的人，哪怕已经在一起了三年，当他心意突然变化的时候还是难以察觉。
我都已经快忘了曾经有几百个夜晚哭泣着无法入睡，甚至都忘了为何哭泣。
也已经适应了自以为的对你的节奏，早已不会因为一点点偏差心中所想就阴郁。
你却又毫不顾忌的挣脱了我种种新砌的篱墙。你还真是任性啊。
关联QQ真不知道是个好东西还是坏东西。嘴上说着不在意却做不到。
但是你仿佛感受到我的小心眼似的，一晚上都陪我说话。
你不停地说开船好玩要带我去。
我说你玩过一次了下次就不会这么好玩了。
你说：和你一起还是好玩的。
其实你已经很久都不说类似的话了，所以我果然感动了。
然后心中的不甘心和嫉妒，就一点一点像气球放气一样通通泄掉了。
越是这样，越是离不开。
太多太多不经意的温暖，都在事后才发现。恋爱中的成长并不是一开始就一定是正面的，也许会绕些弯路，也许会受点苦，但只要那个人是对的人，最终就一定会互相迁就，学会容忍。为了对方而渴望自己能成长为更棒的模样。

我无法在某个特定的时间回想我们甜蜜的片段，但当我踏过每一条路，它们又都无处不在。

话说我到底哪里来的自信，能够紧紧牵住你的手，一直一直？
我自己也不知道吧。
或许，是记忆里小小的你面对我时微红的脸颊？
或许是柜子里你送的围巾、帽子、热水袋？
或许是电话里那句你绝不离开我？
或许是凤凰小醉时你紧紧护住我有力的胳膊？
或许是你骂我的太年轻？
或许只是今天你和我说了很久的话，告诉我你一整天做了些什么。
总之，我就是知道。

对2014说几句

发表于 2019-07-18 | 分类于 愚蠢的浪漫 |

2014的这本日记。
从头看到尾。
只觉得心疼过去的自己。
初时预备把所有有意思的心情搬到网络上，此刻翻着这个满篇写满了想你想你想见你的少女心事，我竟一点也不想再腾写上来了。
2014年不是开心的一年，
看完《无比美妙的痛苦》我在日记里写：
“正因为美妙的过程太短暂，坠落才成为有意义。
正是因为有预期？
我也有预期啊。预期半年，你的预期还是太长了。”
全球爱耳日我写着：
“今天是全球爱耳日，日常要做好爱耳工作。
今天风很大，我单肩搭着书包从外婆家出来再奔到学校，顿时头发飞扬，各种凌乱。
今天上课我不停地告诉自己要专心但一下课就放纵的走心。
今天下午去听冯蕾CC的选修3-3了，六点四十的时候冲到食堂买了个鸡蛋，出来遇见SKY和凉白开，事后无比后悔没有问他们你有没有去开会。如果有的话就可以把鸡蛋给你了。
今天我那双粗糙的手又开始起皮了，幸好你没看见。
今天膝盖不小心磕着桌角，刚刚发现原来有点淤，难怪一直阴疼阴疼的。
昨天我还淡淡地感觉没有你也没什么了不起，反正你也没那么那啥我了。
今天我还是好想你。”

诸如此类，变着花样的描述我如何把你放在心上，刻在脑里。
如果你不喜欢我，你怎么可能看得下去呢。
现在的我，都觉得这些心情太过细碎，不愿读下去。读来读去，都是没有好好学习的青春，和血淋淋的天真。
我记录我小黑丢失的时候给你打电话，你说没关系，人在就好了。可你不知道，许多东西，丢了就是丢了。
我记录听说来的你曾醉酒打我电话，却没拨通，因而难过许久。这种事，你自己应该都不记得。
我还记录那个失而复得的五月，你发来的每条短信，我尤其记得那个夜晚，我蜷缩在外婆家的木床上，一会儿哭一会笑，享受着那个学期许久不曾感受到的来自你信息的提示音——一条又一条，不再间断。
你每条令我舍不得读完的信息，安静燥热的夏夜，构成了我那年最难忘的夜晚。
你说：
  “——每个人还是一直都有解不开的心结啊
  ——你是我的心结。”

你说你最大的缺点是心思过于缜密，拥有复杂的内心世界。你说你一直幻想自己生活在乌托邦，希望事情朝着你希望的方向。你说你做事情总是带着极强的感情性，如果一件事情已经失去了它原本应该被赋予的人情世故，这件事情在你眼里就已经失去了意义。
你感谢我在那段时间给了你深刻的回忆。你说你讨厌自己，因为你，放不下。你说和我在一起念稿子，商量如何分段，聊着最近的事情，才知道多么开心。
你那时候以为你给我发完这些信息我们就不会再有交集，所以你说希望我拥有我想要的明天。

没想到吧，那时候的我，狡黠的利用了那一晚你的不舍，把你捉了回来。
那时候的我还自信，只有我是你的太阳花。
如今看来，太多痕迹，都是被强行留下。
如果你再狠心一点，如果我不执着一点，我踏马现在就不用苦心研究怎么做一个博客，怎么排版，怎么做回自己。我是说，整天对着电脑，我真的快瞎了。

好吧，所以往事可回味，不必追忆。
我每完成一篇博文，心里某块，好像都会回来一点点。
所以2014这个本子，别的多余的重复的心事，就让她们永远的留在17岁吧。

2014年8月9日（闭眼）

发表于 2019-07-18 | 分类于 时间都知道 |

下午我们从水洞下来，在要卖薯条那条店里休息。
我要了一杯西瓜刨冰，碎碎吃着，你趴在玻璃桌上，很困。
我一边用奶色小勺戳着大块的冰团，一边絮絮叨叨地在你耳边碎碎念。
我想到什么说什么，还唱歌，自言自语。

店外边刺眼的阳光预兆着不久而来的闷热大雨。并没有蝉叫声，有蜻蜓低低飞过。

我啊，虽然天气很热，自己眼皮子打着架，但是真的心里很惬意。

且不论你听见了我说的多少话。

也许十年后，你是躺在我们共同选择的红木乔床上，盖着薄薄的纱被，睡意浓浓又努力捕捉着我的声音。
我不会把床帘遮得严严实实的，我要摇起半扇窗户，让大自然的声音和我的声音相知交融，为你的午休带去清凉的气息。
然后我会翻过身，对着你的脸，很小心地摸摸你刚理过的头发，无奈但一点也不失落的呢喃：
好吧，午安。

你看，你对我的好，与我对你的在乎。本质同样深刻只是形式稍许不同。可你干嘛，老是故意看不见。
这应该算爱了吧。

2014年6月10日（笑）

发表于 2019-07-18 | 分类于 时间都知道 |

我在听胡夏唱的同桌的你。
特别喜欢这首歌改编后的伴奏。
很多钢琴元素，应该说基本上都是琴声。
你有没有注意过？
每次我听这首歌，缓缓的琴声好像潺潺的小溪，又好像少女的低语，周小栀和林一年轻美好的脸庞灿烂地浮现在眼前，伴随着从心底流淌出的乐音，我似乎不再盘腿坐在寝室的小床上了，而是整颗心跟着周小栀洁白透明的脸庞飞到了无垠的白云蓝天之中。
细细碎碎的回忆洒满青春的肩头，在一片温暖之中，她朝他伸出了手，眼神是那般坚定。

那一刹那，整个世界都没有了关系。

2014年2月26日（晴）

发表于 2019-07-18 | 分类于 时间都知道 |

我看着你走进教室，从第一组走到第四组，又拎着垃圾桶走出教室，往科技楼方向出去。我透过班级小小的窗玻璃和以前我们做操时那棵核桃老树叶片间的缝隙，目光紧紧追随着你。

这时是七点。各班嘈杂的新闻联播声音却丝毫不影响楼下操场的寂静。
把头迎着微风探出窗外时，明显感受到。

忽然手机震动，我从上往下阅读信息，发件人还没出来。
“好无聊啊。没人上课。”
天呐，或许会是你么？我的心强烈地跳动起来，和班里的喧嚣相比格格不入。
“电杆树不想你电杆树不想你”我心里一遍一遍默念着。
就像无数个空洞的夜晚在心里一遍一遍念着那三个字。你的名字。

唉。
是别人。
不是你。

那一瞬间，我仿佛回到初三，每天明明很在乎你的短信，却又故意总要等半小时才回复你。可惜这次是我单方面的。

三毛说：要爱自己。

2014年2月23日（小雨）

发表于 2019-07-16 | 分类于 时间都知道 |

我从满教室的人等到只有两个人，
从亮堂堂的白炽灯光等到漆黑一片的昏暗，
从前排凌乱的叠卷等到卷子都被收到抽屉里，
从窗下叽叽喳喳三两成群的走读生等到路灯下寂静的一片落叶，
从满心盈斥的兴奋等到不可压抑的悲伤，
从满腔溢出的期待等到死灰般的失望。
我就这么站在窗前，对着窗玻璃反射的自己龇牙咧嘴。
一边告诉自己他绝不会抬头注意二楼的教室，一边揪着心地紧张着每一个牛仔黑衣的男孩。

Hey，我真的......
我不介意了，你的雄心，你的壮志。
我不在乎了，你的话，几分能实现，几分不能。
我不痛苦了，你总也不亮起的昵称，总也不响起的铃声。
我只是想着，我要看到你。要你活得好好的，要你笑着，要你有那么一天，会发现我藏在你手机背面的纸条，在还喜欢着我的一天。


为什么呢，为什么呢。


我愿意回到饱受饥饿的年代，至少那时，一个眼神，就是终生。


老天，让我每天都能看到你吧。
