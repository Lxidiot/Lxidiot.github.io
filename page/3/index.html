异地恋

发表于 2019-10-25 | 分类于 只道寻常 |

2015年十二月，寒假即将来临的日子。
大学的第一个长假期，R智和我回家的日期却无法统一。
R智的学校1月9号就放假，而我的学校1月16日。
这就意味着，1月13日，我的第一个在异地的19岁生日，只有我自己。

R智心中想留下等我，但是我如何能这么自私？不是一天两天，整整
八天，又如何让他和父母推托？
我一直催他买票，从不提及要和他一起回去。R智因为我的不挽留，显得十分不开心。
晚上正忙着部里的事情，对话框忽然亮了起来。点开，果然是他。

“我们马上就要异地恋了。”

这郑重的语气给我吓一跳。
你要出国留学了？要转学了？要去哪个离我很远的省当兵了？被老师慧眼识中要带你去哪学习交流了？一时之间，竟然各种稀奇古怪的想法冒了出来，又一下想到室友和非洲白的寂寞境地，想到今天一个人下晚自习走在黑洞洞的小路上时脑子里充满你的画面，想到自己一个星期才能见你一次其实觉得好不够的贪心，突然就心跳加速了，却仍调笑着回过去带了个期待的表情：

“你要去海外了？(*・０・)”

“我要坐飞机回家了。”

所以你就是这么逗，要不是我心脏好，不定哪天就被你给吓出心脏病来。

“相隔一千多公里。”
你企图继续插刀我。

八天。八天如果你留下了，或许我们能天天见面。但我们年轻啊，我们的未来还那么长。
我不想你父母等你等太久。
这一种对孩子的盼望，是经不起打击的。总有人等着帮你铺上最暖和的被褥，为你晒最柔软的棉絮，备着你最爱吃的菜肴。他们经过你房间时的怔仲和心里的落差，从不会告诉你。
所以我要你回去。

R智是个敏感兴致的人。最近他忙，忙到很少抒发情怀，直到今天，我又看见了我的R智。
天命里，我们一定是分不开的。
携手，一起走。

关于吵架

发表于 2019-10-24 | 分类于 只道寻常 |

有一次和R智吵架，我非常生气，而且是很闷很闷的那种。
好像是因为要出去玩，但是R智显得兴致聊无，也不配合我，恰好那段时间又对我些许冷淡，加上自身刚选上学校部门的一个部长，每天很多杂碎的事情，很烦躁，所以觉得R智变了，变心了等等云云……
到了周末，本来R智说好要来学校看我，结果临时变卦，有事耽误，又说办完事来看我，办完事其实不过下午四点左右，马上动身还是可以来见我。但R智直到五点才给我打电话，然后说晚了，要第二天再来。
当时我就特别生气。那一天我早上就准备好了随时见他，作业也写不进去，裙子试了又换，午饭一直没吃，等着他来，结果却等到这样一个电话。
更让我难受的是，电话里的我已经语气很不好，甚至要求R智立马过来看我，R智却依然以太晚来不及为借口，无所谓般。
我都快要哭了。想见一个人的时候，翻山越岭，徒步走去也要见，更何况是公交车都还有许多班的下午五点？
分明就是不想见。是嫌麻烦。是不care。
刹时我就忍不住了，我在电话这端一口气说完：“好，你不愿意过来；我过去，让你知道，没有晚不晚，只有想不想。”然后挂了电话。
当时拎了个包就出去坐公交。
坐在车上眼泪一个劲地淌啊，想来想去，R智就是变的淡了，既然我已经不是他的菜，那就不要阻碍他幸福……
于是我揣着说分手的沉重心情，18：00站在了他的校门口。
MD可恨的是，他又让我在他校门口登了将近二十分钟。本来我还犹豫要不要这么冲动，可是这么一闹，心里立马凉飕飕的了。
且不说我一个女生贱兮兮地跑来找一个不愿见自己的人，结果颠了一个多小时，人家还让你继续等着，这其中哪里还有爱呀？
R智出现的时候，还是一贯的云清风淡姿态。
他总有这种能力，把周遭与自己隔绝开。
我垮着脸：
“分手吧。”
R智：
“分什么手啊，走，吃饭去，想去哪里吃？”说着伸手过来揽我。
我打开他，自顾走在前面。
“R智，我现在站在这里，就是向你证明，这个时间，一点也不晚。”
他继续嬉皮笑脸:“好啦好啦，知道啦，你还在生气呀？别生气了嘛。”
我有时候也是不懂他到底哪里来的与生嘻哈感，怎么能就这么厚脸皮呢？
“你和我出去玩，我问你去哪里，能不能认真回答我？”
“我早就说过了，随便你呀，你想去哪儿我们就去哪儿。”
“可是你这个随便也太随便了！有些事我要同你商量的，你也要配合我啊！”
“但是我真的随便你呀，我自己而言，我其实是哪儿也不想去的。”
“好，你哪儿也不想去，你就别去啊！我自己和其他同学去玩！”
“可是我想陪你去啊！”
“没关系，我觉得如果你不想去，那我和你一起出去玩就没有意义。”
“对你而言没有意义，但是对我来说，陪你一起出去旅行，就是我的意义。”

走着的脚步忽然就放慢了，马路上的喧嚣声忽然就全都消失了。
差点忘了他是这样一个人。善于表达，却不善于袒露自我；从容谦让，却也常在别人看不见的地方吞咽孤独。他能看透我，我却不理解他。

后来，我们去吃饭了......
后来，就又打打闹闹笑呵呵了......
再后来，我回想起这一天的情景，我问他：
“喂，你知道很多情侣都是吵架的时候分手的吗？”
“嗯？”
“所以那一天，我那么表现严重地和你说你怎么不生气呢不和我吵呢？”
“你自己都说了情侣不是爱吵架的时候分手么。就是因为你已经很生气了，我更不能和你吵了。我们两个人，总要有一个人是保持清醒的啊。”

这大概，就是这个让我又爱又恨的R智的特属温柔与守护的方式吧。

——2016.10.1

喜欢与爱

发表于 2019-10-24 | 分类于 愚蠢的浪漫 |

随笔

喜欢萌发于每一个心动的时刻。
而爱，大抵为产生“永远不分离”这样的念头时偷驻心里。

喜欢是不能累积成爱的。
有些人从青涩的学生时代起喜欢一个人，但却喜欢的浅淡，或许工作时间同学聚会会恍觉自己仍喜欢某某，但这美好的念头也就转瞬即逝了。

喜欢只能升华成爱。
只有一次比一次更加浓烈的喜欢，每一次见面都比上一次还要加倍的倾慕对方，渴望对方，恨不得他就是自己，自己就是他。这种经得住考验，随着岁月的沉沦而沉沦，像生香蕉久置为熟香蕉，越来越甜，越来越透，最后甜味不会散去，也不会平白消失，若没有在最恰当的时机被良人食之，也只好怀着满满的爱而老去。

喜欢是你在被欺负时，会心疼地在一旁看着，时不时说些求情的话，心里惋惜梦中人儿竟如此狼狈。

只有爱是在别人的手指向你的第一刻就从围观的人里冲出来护你在身后，打不赢也要打，骂不过也要拼命回击，为了你，他会长高，变状，像赴死的战士毫无惧色。

喜欢与爱的关系，是怎么也说不清楚，但大家好像又都明白的。

最是那“只可意会，不可言传”。

有夜晚就有白昼

发表于 2019-10-24 | 分类于 愚蠢的浪漫 |
“每一个雨夜，我都想醒在这里，放心地让自己家的雨盛大地抱紧，让心安静。
雨越大，夜愈静，心更静。
静得听得见，你想我，我想你。”

——摘自《雨点落在瓦背上》


有夜晚就有白昼

夜晚和白昼的存在是一件十分美好的事。
当你正处于胸中盈满大大的喜悦心情之际，再闪亮光滑的大理石地面、再熠熠生辉的水晶大吊灯、再价值不菲的南非大钻石，都无法最真实最原始地表露出你想要和每个人拥抱的兴奋与激情。一切物质的东西，只会传递出其冷冰冰的本质。

只有那大片混着青草泥土香的阳光和清晨未散的玉洁露珠能真切地感受到你心中喷张的血脉。待金色的光芒看羞了花朵的脸颊，那无从言表的高兴也自然而然地从你心房溢出，纯真地献给这白昼。

当你孤寂时，你不需要着急掩盖周身扩散的难过，你或许会想一个人静静地坐一会儿。
于是你在万籁俱寂的夜晚，独自爬上顶楼的天台，抱两罐啤酒，望着远处明明灭灭的灯光，晃荡着悬空的双腿，想自己无人可倾诉的心事。

这时的天空从这头黑到那头，分不出浓墨淡墨，模糊了远山的轮廓。好似天地间就只有这一种颜色了，伴随着深深的难以察觉的叹息。

你就这样坐在夜空下，听心里的孤单无助地流淌，流入身后的葡萄架，流进身旁自己孑影里，流进这寂寂的夜空。你恍惚不是一个人，因为有夜晚，陪你一起孤单。
当你渐渐困意袭来，脑袋昏沉得再没心思来想那些缠人的心事，你决定回去舒适的小床，睡觉。
小心翼翼地下楼梯时，像是落下了什么似的失落感让你鬼使神差地再次抬头望去，却一瞬间呼吸都要停滞——不知何时，一颗颗发光的星星高高挂起，夜空有了繁星的拥抱，变成了浩瀚而盛大的星空。
这样一想，夜晚和白昼是相生相交的。
夜晚大概暗恋着白昼，白昼大概是夜晚的恋人。他们絮絮低语的情话，早已化
作芬芳的蔷薇和微笑的月亮。
