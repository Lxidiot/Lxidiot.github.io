春2016.3.12

发表于 2019-07-24 | 分类于 时间都知道 |

  今天是植树节，我记得我假期的时候还在家里抱着手机看了许多植树节可以免费参观的景点，还小心翼翼的用最喜欢的一支圆珠笔标记在爱格杂志送的小本子上。
  结果今天我泡了一天的图书馆，没有跟你聊过一句话，我也不知道你在忙什么。
  我现在会去听一些很弱智的电台节目，主播甜的发腻的声音透过耳机传到耳膜：“如何成为一个优质女友？”“聪明的女孩在恋爱中都是这样做的”……
  有时候也会抓住一些有用的关键词，比如男人是金星的，不要把女人那套思维尝试与他共享，比如要包容他，no zuo no die，比如他玩游戏，如果他能自控，你就不必插嘴管他……
  晚上和妈妈通了半小时的微信视频，我故意告诉他，明天你会来看我，于是我们聊起了这个话题，又是我妈惯用的方式，借他人的例子来暗示我，她说隔壁姐姐找的老公，在她临产两个月这么重要的时刻，还只会打游戏，不愿意每天晚饭后陪她出门散步，洗碗做饭都是姐姐来。我听后很气愤，这样的丈夫，不成熟，不尽责，姐姐结婚时是如何考虑的？
  “刚开始，你姐姐和她老公还谈恋爱时，她老公要和她分手的，可是你姐姐喜欢他，和她妈妈说不愿意分手，她妈妈就闹，硬是不同意他两分手，后来，他们才能走到结婚。”
  “也就是说，你姐姐明知道他是这样的人，却被过多的喜欢蒙蔽，以至于现在要受这样的委屈，也无法可施。可是你想想，你姐姐多年轻啊，往后还有那么多日子，怎么办呢？”
  “所以说，找一个对自己好的男孩子，是多么重要啊。”
  我没搭话，最后，我妈说：
  “女儿，你记住，第一，女孩子要矜持；第二，女孩子对男孩子的爱，不要超出男孩子对女孩子的爱。”
  我笑着，在手机视频的这一端，看似坚定的点头。

  是了，坚定，只是做出来给别人看的坚定，这个头，其实点的很空洞。
  我心里，确是在不断加厚着你对我好的深度。可是我们也没有办法否认，我们的念想和行为随着时间和年龄，在发生着或多或少的改变。
  你掌控着你想要的，我独在高台一侧惶恐。
  这是一种假惶恐。
  明知道高台下是柔软的会稳稳接住自己的棉花，却还是会惶恐。
  否则也就不是这个年纪的爱情了 。
  我说自己点头的空洞，只因妈妈这些话，对我而言，早已没有意义。怪我遇见的人是你，放荡不羁爱自由的你，桀骜不屈执念深的你。你老说你为我改变许多我不了解，可事实上你改变了多少，你就改变了我多少。我早已被你给予的温暖和想象中自我构造的梦境所埋葬。爱着的时候，嘴上说着计较，做起来只会更多。
  我真的从未否认你对我的真心，我只是在同这个年纪的大多数女生一样，在担心着，你对我的真心，维持了三年、四年、五年，能否维持到地老天荒。
  可是电台里教我，不能用自己的思维去和你交谈，所以这样的话，我一次也没有认真的追问过你，我还悄悄地把这情绪藏匿。我也在等待着，哪一天，我不会再想问这样的话。
  未来始终是不可预料的，棉花里面说不定哪天就藏了尖锐细长的硬刺，跌入即死。
  只是你说你明天要来看我，我说是不是梦话，你说我是不是傻逼，你说你说了会来就一定会来。
  我停下摘抄的酸手，手机置于胸前，就这么直直地往后倒去，眼泪蓦地顺着眼角流出来，没有滴下。
  你说我是哭什么呢。咳，我是哭，南京的春天，太冷了。

科巷

发表于 2019-07-24 | 分类于 只道寻常 |

  南京，又称金陵，江南小镇，侬侬温婉，当然，靠近上海，隶属江苏，美食也是少不了的。
  今天，我和R智饭点时就跟着导航七拐八拐的来到了五大吃巷之一的科巷！巷子不大，一眼能笔直的望到尾端。科巷夹在两排老梧桐之间，四处也并没有高高的摩天大楼，一家挨着一家的美食店就这么怡然的停靠在道路两旁。
  天呐，我一看这架式，兰州拉面、滑蛋饭、石锅饭、串串香、北京烤鸭、老南京口味……口水都已经在嗓子眼打转了，拉着R智，我们兴奋的寻觅着准备大干一架的店面。
  这些一间间老城区味十足的小店，没有狮子桥那么繁华的地势和老门东那么华丽的宣传，科巷外表看上去是平凡、朴素的，没有排得长长的队伍和五花八门的嘈杂，原本还以为科巷的人会少些，可走近店面仔细一看，不论大小店面内却是另一番天地，叫号的，买单的，吃饭的，人声鼎沸、座无虚席。
  最终我和R智进了一家名叫“鸿福面馆”的小店。
  R智点了一份小碗的毛肚面。我四下粗粗环顾一圈，开始撒娇：
  “…我不想吃面嘛。”
  “那你等我吃完这碗，我们再去吃其他的。”
  于是R智带着他香气四溢的毛肚面上二楼坐下后，在我对面开吃起来。
  金灿灿的油汤，放有一点辣油，几片青菜点缀，毛肚镶嵌期间，冒着腾腾的热气，配着R智享受的表情，关键是，我饿了。
  “分我吃一口嘛~”
  “不分。”
  “你怎么那么小气！”
  “不给就是不给，要吃自己再点一碗。”
  “。。。。。。”
  过了一会儿，我屈服了。
  “喂，我也要吃一碗面。”
  “嗯，你去点。”
  “我要吃他家招牌鸿福面！”
  “那你点个全家福吧。”
  “什么全家福，全家福什么！我点鸿福面啊，鸿福面馆鸿福面……”
  边说我就边走远了，R智说什么我也没注意，我说什么R智也没注意，我来到柜台就径直说，老板，给我来一份小鸿福！
  后来……
  鸿福面，58块钱一碗，小份45，以份量足、配料多、口味佳而名扬吃货圈。普通的一碗面适合3至5个人吃，就是小份的推荐人数也是2至四人，店里常常是三两朋友或情侣相约来吃，店里更是粘贴着一份挑战书，店主公然保证，谁一个人吃完一份鸿福面立即免单。
  现在，一个适合2-4人共吃的鸿福面放在我这个心大胃口小的人面前，满满的一大盆……
  我目瞪口呆…..
  早说过了我不爱吃面，完全是凭着店名来选面的，什么也没看，什么也没在意，我说下单的时候怎么这么贵呢，一碗面45…..原来是因为它这么多……
  R智也呆了：“你点了鸿福面？”
  “。。。”
  “我不是让你点全家福吗？”
  “我点的小份鸿福面(委屈)…..”
  “全家福是更小份的啊！”
  R智不说话了，突然，他狂笑起来…
  “还有人比你更蠢吗？哈哈哈哈快吃，我看着你把这碗面吃完哈哈哈…”
  “你看看周围谁不是两个人吃一份，你瞧人家都盯着你的面在看了，不知道的人还以为这个小姑娘多能吃呢哈哈哈哈…..”
  我这才意识到自己是有傻，点单之前都不搞清楚，我也很后悔啊：
  “你怎么这样，我又不知道是这种TAT…..”
  “哈哈哈哈哈哈哈……”
  “谁让你不跟我一起去点！T^T”
  “哈哈哈哈哈哈哈….”
  “你有完没完！”
  “没有，哈哈哈哈哈哈…..
  快吃，不能浪费呀我给你一个小时，慢慢吃哈哈哈哈哈哈  哈…..”
  。。。。。
  最后，那碗面我当然吃不完，而且我都吃撑了，那碗面还像没吃过的一样…..而R智，这个只会嘲笑我的人，从头到尾把自己的面吃完吃饱以后就不肯吃帮我吃一口鸿福面，他说我自己的选择要我自己承担…..虽然他说的是实话……
  出店的时候我简直想把脸塞到门缝里。
  我对不起爸妈，对不起那45块钱，对不起科巷的其他小吃！
  这一天将是我人生里无比灰暗的一天，还能说什么呢，快点洗洗睡明天少吃点。
  R智临走前说“下次认不认真听我说话了？自作自受！”
  哼！！！

愚蠢的浪漫

发表于 2019-07-24 | 分类于 愚蠢的浪漫 |

  遇见一个你很爱很爱的人，恰好也很爱很爱你，是怎样的感受呢？
  我原来以为是烟花绽放时，我同这漫天的美景被恰好站在那儿的你尽收眼里，心中随着“噼里啪啦”的烟花声盛放出一朵又一朵的灿烂花朵。
  后来，当他只把小孩子的那面向你展露，为你和别人的一声打闹而吃醋时，你就知道，你没有找到最正确的那个答案。
  我于是以为，那种感受应该是我在春风明媚的阳光里坐在装饰得最好看的小店里吃到了我最爱吃的土豆，满腔的满足感与风和日丽。
  可是后来，当他陪着你一起，万里无云的蓝天，喧嚣吵闹的放学，张扬放肆的青春，不嫌弃地陪着你站在那家最好吃的摊位前等着你心心念念的洋芋出锅，你忽然发现，原来你的心，还可以更膨胀一点。
  我最终以为，那就应该是像何以琛和赵默笙牵着小何照在树林里聊天那样，让人生，变得完整。
  直到今天，我发现，我又错了。
  当R智包裹着我冰冷的手紧紧护在他兜里时；当我们散步在傍晚的细雨凉风中时；
  当我们都因不忍破坏这温存的时光而都静谧时；
  当我回到学校收到短信时——来自你的短信：“因为想念才会多想”。
  我什么，都无法写下。
  因为这感受，无法表达。

失去与得到

发表于 2019-07-22 | 分类于 只道寻常 |

  成长是一种蜕变，失去了旧的，必然因为又来了新的，这就是公平。
  ——三毛

  所以十三岁的我失去了羞羞答答的你，得到了专注的自己。
  所以十五岁的是失去了大大咧咧的自己，得到了眼里满是我的你。
  所以十七岁的我失去了许多咸咸的泪水，得到了更加心疼我的你。
  所以十八岁的我失去了爱吃醋的你，同时得到了学会贴心、稳重的你。
  如果三毛的话始终会应验，那么我祈愿，将来无论我失去怎样的你，都能永远拥有爱我的你。
