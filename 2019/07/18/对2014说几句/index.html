对2014说几句

发表于 2019-07-18 | 分类于 愚蠢的浪漫 | |

  2014的这本日记。
  从头看到尾。
  只觉得心疼过去的自己。
  初时预备把所有有意思的心情搬到网络上，此刻翻着这个满篇写满了想你想你想见你的少女心事，我竟一点也不想再腾写上来了。
  2014年不是开心的一年，
  看完《无比美妙的痛苦》我在日记里写：
  “正因为美妙的过程太短暂，坠落才成为有意义。
  正是因为有预期？
  我也有预期啊。预期半年，你的预期还是太长了。”
  全球爱耳日我写着：
  “今天是全球爱耳日，日常要做好爱耳工作。
  今天风很大，我单肩搭着书包从外婆家出来再奔到学校，顿时头发飞扬，各种凌乱。
  今天上课我不停地告诉自己要专心但一下课就放纵的走心。
  今天下午去听冯蕾CC的选修3-3了，六点四十的时候冲到食堂买了个鸡蛋，出来遇见SKY和凉白开，事后无比后悔没有问他们你有没有去开会。如果有的话就可以把鸡蛋给你了。
  今天我那双粗糙的手又开始起皮了，幸好你没看见。
  今天膝盖不小心磕着桌角，刚刚发现原来有点淤，难怪一直阴疼阴疼的。
  昨天我还淡淡地感觉没有你也没什么了不起，反正你也没那么那啥我了。
  今天我还是好想你。”

  诸如此类，变着花样的描述我如何把你放在心上，刻在脑里。
  如果你不喜欢我，你怎么可能看得下去呢。
  现在的我，都觉得这些心情太过细碎，不愿读下去。读来读去，都是没有好好学习的青春，和血淋淋的天真。
  我记录我小黑丢失的时候给你打电话，你说没关系，人在就好了。可你不知道，许多东西，丢了就是丢了。
  我记录听说来的你曾醉酒打我电话，却没拨通，因而难过许久。这种事，你自己应该都不记得。
  我还记录那个失而复得的五月，你发来的每条短信，我尤其记得那个夜晚，我蜷缩在外婆家的木床上，一会儿哭一会笑，享受着那个学期许久不曾感受到的来自你信息的提示音——一条又一条，不再间断。
  你每条令我舍不得读完的信息，安静燥热的夏夜，构成了我那年最难忘的夜晚。
  你说：
   “——每个人还是一直都有解不开的心结啊
   ——你是我的心结。”
  你说你最大的缺点是心思过于缜密，拥有复杂的内心世界。你说你一直幻想自己生活在乌托邦，希望事情朝着你希望的方向。你说你做事情总是带着极强的感情性，如果一件事情已经失去了它原本应该被赋予的人情世故，这件事情在你眼里就已经失去了意义。
  你感谢我在那段时间给了你深刻的回忆。你说你讨厌自己，因为你，放不下。你说和我在一起念稿子，商量如何分段，聊着最近的事情，才知道多么开心。
  你那时候以为你给我发完这些信息我们就不会再有交集，所以你说希望我拥有我想要的明天。

  没想到吧，那时候的我，狡黠的利用了那一晚你的不舍，把你捉了回来。
  那时候的我还自信，只有我是你的太阳花。
  如今看来，太多痕迹，都是被强行留下。
  如果你再狠心一点，如果我不执着一点，我踏马现在就不用苦心研究怎么做一个博客，怎么排版，怎么做回自己。我是说，整天对着电脑，我真的快瞎了。

   好吧，所以往事可回味，不必追忆。
  我每完成一篇博文，心里某块，好像都会回来一点点。
  所以2014这个本子，别的多余的重复的心事，就让她们永远的留在17岁吧。

# 2014
2014年8月9日（闭眼）
2015年8月29日（初来金陵）
