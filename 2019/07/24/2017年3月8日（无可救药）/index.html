2017年3月8日（无可救药）

发表于 2019-07-24 | 分类于 时间都知道 | |

  很久没有写点什么了，我现在越来越多的腹语，当时当刻其实我心里已经闪过一百种说话方式和事情发展走向，但是最后，都不会说出来。
  这个季节学校里的二月兰正盛放，大活前面，文德后面，整片整片铺满的紫色。很好的景色，可惜没有你在。
  我今天擦了你送的那支口红，真的特别好看，颜色很衬肤色，感觉自己变好看了，可惜没有你在。
  我过去很在乎那些小的细节，现在我已经没那么在意，谁和谁走在路上也一直拉着手，谁与谁相伴图书馆，谁对谁情深意切的说着想念，谁放学等在哪个路口陪她去吃饭。
  我老是回忆起你曾为我辗转反侧的夜晚，为我发的高烧，为我存钱买的礼物，为我改变的温柔。
  今天我们班委去听辅导员讲课，我们宿舍的去晚了，去的时候，单出了一个人要独自坐在教室的最后。那个人是Q狍子。狍子的男朋友坐在前面，旁边的位置坐着另一个男生，我朝那儿指了指，让狍子去和男朋友坐，狍子不好意思，推攘着说没关系就一个人坐在我后面。狍子男朋友其实一直往这边望着的，我说话的声音很大，他也听见了，但他只是看着Q狍子的反应，并未起身，也没有说话。后来，辅导员来了，也就这样坐着了。
  这也只是小事，坐哪儿都一样，大家没觉得有什么不妥。
  只有我，心里默默的想着我的R智。
  如果是你啊，你一定会让我过去和你坐，或者，陪我一起坐在后面，在一起。
  就是这么笃定。可是如果我这么和你说我的想法，那你大概又要笑得玩世不恭：“哈哈你想多咯，我哪里会管你！”。
  你也是爱心口不一的。事实和重要的事，你想告诉我就会告诉我，不想告诉我，就会把牛皮吹成耕地的牛，反正不用太厉害，能唬住我就好。我又老是上你的套。
  同一个城市，不同的学校，每周见一次，我觉得太少。
  我喜欢每天都能看见你，渴望每秒能听见你的声音，听见你高兴的唱出来：“我有一个L小西，我从来也不理……”。
  瞧，歌词都在损我。
  那你有强迫症，不愿意周一到周五我去找你或者你来找我，周末要看你心情，看你睡眠情况，也许头晚上随口说了一句“明天来看你”，第二天突然有事，或者没睡醒，直到我一两点打电话过去喊饿了，才知道，你不来了。
  这种类似的希望到失望到预测到失望还是忍不住绝望失落的感觉，我已经描述得够多了。
  我在尝试，不受你摆布，既然你可以想不说话就不说话，我也可以。虽然我总是失败。
  每次写这样的文字，心里就揪着又难受一遍，又要哭，又想拿起手机对你说胡话。我似乎在由后往前一项一项的进步，可以忍住不看手机，忍着眼泪不掉下来，却依然忍不住，让自己忽视，让自己不这么难受。  好像想你， 倒成了罪孽。

# 2017
