蓝鲸不冷

发表于 2019-07-24 | 分类于 只道寻常 | |

  十二月的蓝鲸，寒风里的温馨。
  R智来到我们学校，我带他去南门美食一条街吃午饭。

  R智的学校那可谓一个高大上啊，那是一个新修没多久的大学。如果跟我们学校比较，他简直就是贵族公子下乡。
  不过正因为他们学校讲究规划，环境优美，像我们这种小吃巷他们是没有的，什么梅花糕、双拼土豆豆腐、灌蛋饼、花甲、热气腾腾的鱿鱼……在我们南门市场内走一圈，简直到了吃货的天堂！

  我和R智一人一份桥头排骨，一杯手工酸奶，面前放着一盘宇宙无敌超级好吃的炒饭（其实是我和室友们取的名字，但真的很好吃，老板人很好，饭菜也很实惠^o^）。
  十二月的城市，本来是冷的。
  面前这两份炒饭飘出来的香味和徐徐上升的热气却盖过了店外的寒冷，一时我竟忘记了，自己在南京的冬天里。

  我笑眼盈盈地看着R智：
  “你说，你幸不幸福？”

  “……

  嗯，幸福。”

2015.12.20 周日
