科巷

发表于 2019-07-24 | 分类于 只道寻常 | |

南京，又称金陵，江南小镇，侬侬温婉，当然，靠近上海，隶属江苏，美食也是少不了的。
今天，我和R智饭点时就跟着导航七拐八拐的来到了五大吃巷之一的科巷！巷子不大，一眼能笔直的望到尾端。科巷夹在两排老梧桐之间，四处也并没有高高的摩天大楼，一家挨着一家的美食店就这么怡然的停靠在道路两旁。
天呐，我一看这架式，兰州拉面、滑蛋饭、石锅饭、串串香、北京烤鸭、老南京口味......口水都已经在嗓子眼打转了，拉着R智，我们兴奋的寻觅着准备大干一架的店面。
这些一间间老城区味十足的小店，没有狮子桥那么繁华的地势和老门东那么华丽的宣传，科巷外表看上去是平凡、朴素的，没有排得长长的队伍和五花八门的嘈杂，原本还以为科巷的人会少些，可走近店面仔细一看，不论大小店面内却是另一番天地，叫号的，买单的，吃饭的，人声鼎沸、座无虚席。
最终我和R智进了一家名叫“鸿福面馆”的小店。
R智点了一份小碗的毛肚面。我四下粗粗环顾一圈，开始撒娇：
“...我不想吃面嘛。”
“那你等我吃完这碗，我们再去吃其他的。”
于是R智带着他香气四溢的毛肚面上二楼坐下后，在我对面开吃起来。
金灿灿的油汤，放有一点辣油，几片青菜点缀，毛肚镶嵌期间，冒着腾腾的热气，配着R智享受的表情，关键是，我饿了。

“分我吃一口嘛~”
“不分。”
“你怎么那么小气！”
“不给就是不给，要吃自己再点一碗。”
“。。。。。。”
过了一会儿，我屈服了。
“喂，我也要吃一碗面。”
“嗯，你去点。”
“我要吃他家招牌鸿福面！”
“那你点个全家福吧。”
“什么全家福，全家福什么！我点鸿福面啊，鸿福面馆鸿福面......”
边说我就边走远了，R智说什么我也没注意，我说什么R智也没注意，我来到柜台就径直说，老板，给我来一份小鸿福！
后来......
鸿福面，58块钱一碗，小份45，以份量足、配料多、口味佳而名扬吃货圈。普通的一碗面适合3至5个人吃，就是小份的推荐人数也是2至四人，店里常常是三两朋友或情侣相约来吃，店里更是粘贴着一份挑战书，店主公然保证，谁一个人吃完一份鸿福面立即免单。
现在，一个适合2-4人共吃的鸿福面放在我这个心大胃口小的人面前，满满的一大盆......
我目瞪口呆.....
早说过了我不爱吃面，完全是凭着店名来选面的，什么也没看，什么也没在意，我说下单的时候怎么这么贵呢，一碗面45.....原来是因为它这么多......
R智也呆了：“你点了鸿福面？”
“。。。”
“我不是让你点全家福吗？”
“我点的小份鸿福面(委屈).....”
“全家福是更小份的啊！”
R智不说话了，突然，他狂笑起来...
“还有人比你更蠢吗？哈哈哈哈快吃，我看着你把这碗面吃完哈哈哈...”
“你看看周围谁不是两个人吃一份，你瞧人家都盯着你的面在看了，不知道的人还以为这个小姑娘多能吃呢哈哈哈哈.....”
我这才意识到自己是有傻，点单之前都不搞清楚，我也很后悔啊：
“你怎么这样，我又不知道是这种TAT.....”
“哈哈哈哈哈哈哈......”
“谁让你不跟我一起去点！T^T”
“哈哈哈哈哈哈哈....”
“你有完没完！”
“没有，哈哈哈哈哈哈.....
快吃，不能浪费呀我给你一个小时，慢慢吃哈哈哈哈哈哈
哈.....”
。。。。。
最后，那碗面我当然吃不完，而且我都吃撑了，那碗面还像没吃过的一样.....而R智，这个只会嘲笑我的人，从头到尾把自己的面吃完吃饱以后就不肯吃帮我吃一口鸿福面，他说我自己的选择要我自己承担.....虽然他说的是实话......
出店的时候我简直想把脸塞到门缝里。
我对不起爸妈，对不起那45块钱，对不起科巷的其他小吃！
这一天将是我人生里无比灰暗的一天，还能说什么呢，快点洗洗睡明天少吃点。
R智临走前说“下次认不认真听我说话了？自作自受！”
哼！！！

# R智
