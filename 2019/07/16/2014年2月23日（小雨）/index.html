2014年2月23日（小雨）

发表于 2019-07-16 | 分类于 时间都知道 | |

  我从满教室的人等到只有两个人，
  从亮堂堂的白炽灯光等到漆黑一片的昏暗，
  从前排凌乱的叠卷等到卷子都被收到抽屉里，
  从窗下叽叽喳喳三两成群的走读生等到路灯下寂静的一片落叶，
  从满心盈斥的兴奋等到不可压抑的悲伤，
  从满腔溢出的期待等到死灰般的失望。
  我就这么站在窗前，对着窗玻璃反射的自己龇牙咧嘴。
  一边告诉自己他绝不会抬头注意二楼的教室，一边揪着心地紧张着每一个牛仔黑衣的男孩。

  Hey，我真的……
  我不介意了，你的雄心，你的壮志。
  我不在乎了，你的话，几分能实现，几分不能。
  我不痛苦了，你总也不亮起的昵称，总也不响起的铃声。
  我只是想着，我要看到你。要你活得好好的，要你笑着，要你有那么一天，会发现我藏在你手机背面的纸条，在还喜欢着我的一天。

  为什么呢，为什么呢。

  我愿意回到饱受饥饿的年代，至少那时，一个眼神，就是终生。

  老天，让我每天都能看到你吧。
